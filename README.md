# git-practice

### git方法紀錄 ###
* 在git先創repo:
  1. git clone下來
  2. 在裡面建立project..寫code
  3. 最後git add .
  4. git commit 
  5. git push
* 已經在local有repo:
  1.git init
  2.git add .
  3.git commit
  4.git remote add origin https://github.com/yourusername/your-repo-name.git
  5.git push -u origin master
* remove origin 方法
  1.git remove rm origin
* remind
  1.每次再更改code先git pull，確保資料最新
